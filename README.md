## Task Name
Multithreaded Chat Application

## Description
This project is developed as part of the CODTECH Internship.
It is a Java-based client-server chat application that allows
multiple clients to communicate using multithreading.

## Features
- Client-server communication
- Multithreading support
- Real-time message broadcasting

## Technologies Used
- Java
- Eclipse IDE
- Sockets
- Multithreading

## How to Run
1. Run ChatServer.java
2. Run ChatClient.java in multiple consoles
3. Type messages to communicate between clients

## Output
Messages sent by one client are received by all connected clients.
